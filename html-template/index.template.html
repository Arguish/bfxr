	    Keyboard shortcuts: Ctrl+C / Ctrl+V to copy/paste (cmd will work on OSX instead of ctrl), Enter to play current sound.	    
	    

	    Source code is here on github.  As with as3sfxr, this is released under Apache Licence 2.0.As with Tom's version, this includes a standalone as3 synthesizer that's easy enough to integrate into other projects.
	    

	    If you experience any stuttering while a sound is synthesizing, if you hit play again it should play through interrupted... .
	    

	    Original concept by the beautiful Dr Petter, and then ported to flash by the darling Tom Vian [this version is based very heavily on his code].  I haven't fully checked backwards-compatibility, but I think things made with other programs should work with this one.  Lots of code snippits pulled from lots of places around the net, and useful suggestions by @docky, @Draknek, @KommanderKlobb, @eigenbom and @GrimFang4.     
	    

	    If you want to see some of the knowledge behind this stuff, check out this short article on sound synthesis by Dr Petter.
	    
	
	    At the moment the file format is not compatible with other versions of sfxr (thoguh this should load their files ok).
